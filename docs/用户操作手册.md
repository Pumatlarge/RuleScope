## 系统概述

企业规章制度检索管理系统是专门为企业设计的文档管理平台，主要功能包括：

- **文档上传与处理**：支持Word文档上传，自动识别文档信息
- **版本管理**：自动管理文档版本，支持历史版本查看
- **版本对比**：直观对比不同版本之间的差异
- **多维度检索**：支持按标题、关键词、日期、类型等多条件检索
- **权限管理**：基于角色的权限控制，确保数据安全

### 角色权限说明

| 角色 | 权限范围 | 主要功能 |
|------|----------|----------|
| 管理员 | 全部权限 | 用户管理、系统配置、所有文档操作 |
| 经理 | 高级权限 | 上传、编辑、查看所有文档 |
| 普通用户 | 基本权限 | 上传、查看文档，编辑自己的文档 |
| 查看者 | 只读权限 | 只能查看文档 |

## 登录与注册

### 登录系统

1. 打开浏览器，访问系统地址（如：http://localhost:3000）
2. 在登录页面输入用户名和密码
3. 点击"登录"按钮

**注意事项：**
- 首次使用系统需要先注册账号
- 请记住您的用户名和密码，丢失后请联系管理员
- 登录成功后会自动跳转到主界面

### 注册新用户

1. 在登录页面点击"注册"按钮
2. 填写注册信息：
   - 用户名：3-50个字符，只能包含字母、数字、下划线
   - 邮箱：有效的邮箱地址
   - 密码：至少6位字符
   - 姓名：您的真实姓名
   - 部门：选择您所在的部门
   - 职位：您的职位信息
3. 点击"注册"按钮完成注册

## 文档上传

### 上传流程

1. 点击"上传文档"按钮
2. 填写文档基本信息：
   - **文档标题**：必填项，建议使用标准命名
   - **所属部门**：选择文档所属的部门
   - **文档类型**：选择文档的类型（管理制度、操作规范等）
   - **分类**：选择文档的分类（人事管理、财务管理等）
   - **签发人**：文档签发人姓名
   - **签发日期**：文档签发日期
   - **生效日期**：文档生效日期
   - **标签**：用逗号分隔的关键词
   - **描述**：文档的简要描述

3. 上传Word文档：
   - 点击"点击或拖拽文件到此区域上传"
   - 选择要上传的.docx或.doc文件
   - 文件大小不能超过10MB
   - 系统会自动验证文件格式

4. 点击"上传文档"按钮完成上传

### 自动识别功能

系统会自动识别文档中的以下信息：
- **签发人**：从文档末尾自动提取签发人信息
- **签发日期**：从文档末尾自动提取日期信息
- **关键词**：自动提取文档中的关键词
- **字数统计**：自动统计文档字数

### 上传注意事项

1. **文件格式**：只支持.docx和.doc格式的Word文档
2. **文件大小**：单个文件不能超过10MB
3. **文件命名**：建议使用有意义的文件名
4. **内容规范**：文档应包含完整的签发信息
5. **权限要求**：普通用户只能上传自己的文档

## 文档检索

### 基础检索

1. 在检索页面输入关键词
2. 系统会自动在文档标题、内容和关键词中搜索
3. 搜索结果会显示匹配的文档列表
4. 可以使用多个关键词进行组合搜索

### 高级检索

#### 多条件组合搜索

- **关键词搜索**：输入要搜索的关键词
- **部门筛选**：选择特定部门
- **类型筛选**：选择特定文档类型
- **分类筛选**：选择特定分类
- **日期范围**：设置上传时间范围
- **签发日期范围**：设置签发日期范围
- **生效日期范围**：设置生效日期范围

#### 搜索技巧

1. **关键词组合**：使用空格分隔多个关键词
2. **模糊搜索**：支持部分匹配搜索
3. **精确匹配**：使用引号包围精确短语
4. **排除搜索**：使用减号排除特定词汇

#### 搜索结果展示

- **相关性排序**：搜索结果按相关性排序
- **高亮显示**：搜索关键词在结果中高亮显示
- **详细信息**：显示文档的基本信息和摘要
- **快速操作**：可以直接查看文档详情

### 搜索建议

系统提供智能搜索建议：
- 输入时自动推荐相关词汇
- 显示热门搜索关键词
- 提供搜索历史记录

## 版本管理

### 版本概述

系统会自动管理文档的所有版本：
- 每次上传新文档创建版本1.0
- 每次编辑文档创建新版本（1.1, 1.2, 2.0等）
- 所有版本都会保留在系统中
- 最新版本会标记为"当前版本"

### 查看版本历史

在文档详情页面可以查看版本历史：
- 显示所有版本的列表
- 显示版本号、上传人、上传时间
- 可以查看任意版本的具体内容
- 可以对比不同版本之间的差异

### 版本信息

每个版本包含以下信息：
- **版本号**：自动生成的版本号
- **上传人**：上传该版本的用户
- **上传时间**：版本上传时间
- **变更说明**：版本变更的简要说明
- **文档内容**：该版本的完整内容

## 版本对比

### 版本对比功能

系统提供直观的版本对比功能：

1. 选择要对比的两个版本
2. 系统会自动分析两个版本的差异
3. 高亮显示新增、删除和修改的内容
4. 提供详细的变更统计信息

### 对比操作步骤

1. 在文档详情页面的版本列表中选择两个版本
2. 点击"对比"按钮
3. 系统会显示对比结果页面

### 对比结果说明

#### 变更类型标识
- **绿色背景**：新增的内容
- **红色背景**：删除的内容
- **黄色背景**：修改的内容
- **无背景**：未变更的内容

#### 变更统计
- **新增行数**：统计新增的文本行数
- **删除行数**：统计删除的文本行数
- **修改行数**：统计修改的文本行数
- **变更比例**：变更内容占总内容的百分比

#### 对比视图
- **左右对比**：左右并排显示两个版本
- **上下对比**：上下显示两个版本
- **统一视图**：在同一视图中高亮显示差异

### 导出对比结果

可以将对比结果导出为：
- PDF格式：适合打印和存档
- Word格式：适合进一步编辑
- HTML格式：适合网页展示

## 常见问题

### 文档上传问题

**Q: 为什么上传文档失败？**
A: 可能的原因包括：
- 文件格式不支持（只支持.docx和.doc）
- 文件大小超过10MB限制
- 网络连接问题
- 文件损坏

**解决方法：**
- 确保文件格式正确
- 压缩文件大小
- 检查网络连接
- 重新准备文件

**Q: 系统无法识别文档中的日期信息？**
A: 系统会自动识别常见的日期格式，如果识别失败：
- 确保文档中包含标准的日期格式
- 可以手动填写签发日期和生效日期
- 检查文档末尾是否有相关信息

### 检索问题

**Q: 搜索结果不准确？**
A: 可能的原因：
- 关键词过于模糊
- 搜索条件太多
- 文档内容格式特殊

**解决方法：**
- 使用更精确的关键词
- 调整搜索条件
- 尝试不同的搜索策略

**Q: 搜索速度很慢？**
A: 可能的原因：
- 网络连接慢
- 数据量大
- 服务器负载高

**解决方法：**
- 检查网络连接
- 使用更具体的搜索条件
- 联系系统管理员

### 版本管理问题

**Q: 如何恢复到之前的版本？**
A: 系统支持版本回滚：
1. 在文档详情页面查看版本历史
2. 选择要恢复的版本
3. 点击"恢复此版本"按钮
4. 确认恢复操作

**Q: 版本对比结果不清晰？**
A: 可以调整对比视图：
- 切换不同的对比模式
- 调整文本显示大小
- 使用导出功能查看更清晰的对比结果

### 权限问题

**Q: 无法编辑某个文档？**
A: 权限限制的原因：
- 不是文档的所有者
- 文档被锁定
- 权限配置错误

**解决方法：**
- 联系文档所有者
- 联系系统管理员检查权限配置
- 确认是否有足够的操作权限

**Q: 无法查看某些文档？**
A: 可能的原因：
- 文档权限设置限制
- 用户角色权限不足
- 文档状态异常

**解决方法：**
- 联系文档所有者授权
- 联系系统管理员调整权限
- 确认文档状态是否正常

### 系统使用技巧

#### 提高检索效率
1. **使用精确搜索**：对重要关键词使用引号包围
2. **组合搜索**：同时使用多个搜索条件
3. **利用筛选功能**：结合部门、类型等条件筛选
4. **保存搜索结果**：对重要搜索结果进行收藏

#### 文档管理优化
1. **规范命名**：使用统一的文档命名规则
2. **及时更新**：定期更新过期文档
3. **合理分类**：为文档设置准确的分类和标签
4. **版本控制**：重要的文档修改前先创建备份版本

#### 权限管理建议
1. **最小权限原则**：只给用户必要的权限
2. **定期审核**：定期检查用户权限配置
3. **权限变更记录**：记录权限变更历史
4. **紧急权限申请**：建立紧急权限申请流程
